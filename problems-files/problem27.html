אוילר גילה את הנוסחה הריבועית יוצאת הדופן:

$n^2 + n + 41$

מסתבר שהנוסחה תייצר $40$ ראשוניים עבור מספרים עוקבים $0 \le n \le 39$. למרות זאת, כאשר $n = 40$ אז $40^2 + 40 + 41 = 40(40 + 1) + 41$ מתחלק ב-$41$, ובוודאי כאשר $n = 41$ אז $41^2 + 41 + 41$ מתחלק בבירור ב-$41$.

הנוסחה המדהימה $n^2 - 79n + 1601$ התגלתה, והיא מייצרת $80$ ראשוניים עבור מספרים עוקבים $0 \le n \le 79$. המכפלה של המקדמים, $-79$ ו-$1601$, היא $-126479$.

נסתכל על משוואות ריבועיות מהצורה:

$n^2 + an + b$, כאשר $|a| < 1000$ ו-$|b| \le 1000$

ו-$|n|$ הוא הערך המוחלט של $n$
למשל $|11| = 11$ ו-$|-4| = 4$

מצאו את המכפלה של המקדמים, $a$ ו-$b$, של המשוואה הריבועית שמייצרת הכי הרבה ראשוניים עבור ערכים עוקבים של $n$, החל מ-$n = 0$.
